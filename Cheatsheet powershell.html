Cheat sheet

Cheat sheet PowrShell

Название параметра, значение параметра

Основные команды
Get-Command (или gcm) Список всех команд
Get-Member (или gm) Свойства и методы объекта
Get-Help (или man) Получает помощь
Get-Help about* Описание справ. категорий
man <NameCommand> -Parameter <NameParameter> Справка по параметру команды
Навигация по ресурсам
Set-Location (или cd) . Тек. дериктория
Set-Location .. Переход на уровень выше
Set-Location ~ Переход в папку тек. пользователя
Set-Location -Path C:\Temp Переход в папку C:\Temp
pwd Вывод тек. дериктории
Работа с файлами
Get-ChildItem (или ls, dir) список файлов и папок
Get-ChildItem -Path C:\* список файлов и папок
Get-ChildItem -Path C:\Users, C:\Temp Содержание 2-х папок
"C:\Users" | Get-ChildItem Содержание C:\Users
Get-ChildItem -Path C:\* -Include *.zip список zip файлов
Get-ChildItem -Path C:\* -Exclude *.zip список файлов кроме zip
Get-ChildItem -Recurse список файлов и папок с вложениями
Get-ChildItem -File список файлов
Get-ChildItem -Directory список папок
Get-Item C:\Temp Получает свойства элемента
New-Item Создание элемента
New-Item -name myFile1, myFile2 Создание файлов myFile1, myFile2
New-Item myFile1 -Path c:\ Создание файла в корне C:
New-Item myFile1 -Value "text for file" создать файл с текстом text for file
New-Item myFile1 -ItemType File/Directory Создание файла/папки
Remove-Item Удаление элемента
Remove-Item C:\Temp\* Удаление элементов в папке C:\Temp
Remove-Item C:\Temp\* -Recurse Удаление элементов включая вложенные папки
Copy-Item file1 file2 Копирование file1 в file2
Move-Item file1 file2 Перемещение/переименование файла или папки
Move-Item -Path C:\Temp -Destination E:\ Перемещение папки Temp с файлами в корень диска E:
Get-Content Содержание файла
Get-Content .\temp.txt Содержание файла temp.txt
Get-Content -Path .\* -Filter *.log Содержание файлов log
Add-Content Добавляет в конец файла данные
Информационные команды
Get-Alias Список алиасов
Get-Alias -Definition get-alias Алиасы для команды get-alias
Get-PSprovider Получение информации о поставщике
Команды форматирования
Sort-Object Сортировка данных
Sort-Object Size -Descending Сортировка данных по убыванию
Format-Table (или ft) Вывод параметров в колонки
Get-Process | ft -Property * Вывод всех свойств
Get-Process | ft -Property ID, Name
Get-Process | ft -Property ID, Name -GroupBy Company
Format-List (или fl) Вывод параметров в строки
Get-Process | fl * Вывод всех параметров
Get-Process | fl -Property ID, Name
Select-Object (или select) Выборка объектов и его свойств
ForEach-Object (или %) Выполняет операцию для каждого элемента вх данных
ls env:path | % {$_.value -split ";"} Разбор строки реквизита объекта
Measure-Object Команда расчета
ls | measure-object -Property Length -sum Размер файлов в тек. папке
Менеджер ресурсов
Get-Process (или gps) Запущенные процессы
Stop-Process Останавливает один и более процессов
Stop-Process -Id 9885 Останавка процесса с id 9885
Get-Service Сервисы на лок. или уд. компьютере
Stop-Service Останавка одного или несколько запущенных служб
Stop-Service -Name AppMgmt Останавка службы AppMgmt
Start-Service Запуск одной или больше остановленных служб
Get-Volume Получает объекты
Get-PSProvider Поставщики услуг
Get-PSdrive Ресурсы поставщиков услуг
Сетевые команды
Test-Connection Отправка ICMP запросов
Test-Connection -ComputerName server02 Отправка ICMP запроса на server02
Test-Connection server02 -Count 1 Отправка 1-го пакета ICMP на server02
Invoke-WebRequest (или curl) Содержимое Web страницы
curl -Uri 'https:\\google.com' Получить содержание страницы google.com
curl -Uri http:\\site.io\file.zip -outfile myfile.zip Скачать файл
Операторы
1..10 | foreach {$_ * 2} Диапазон
-and, -or, -xor, -not, ! Логические операторы
-eq, -ne равно, не равно
-gt, -ge больше, больше или равно
-lt, -le меньше, меньше или равно
-like, -nolike выполняют сопоставление шаблонов с использованием таких символов как *(ноль или более символов) или ?(один символ)
-match, -nomatch регулярные выражения
-contains, -notcontains операторы вложения
-replace Оператор замены с использование рег. выражения
-split, -join Операторы разделения объединения
$d = get-data('25/05/2022') Присваивание даты переменной d
$d = @ {k1='v1';k2='v2'} Словарь
Политика выполнения
Get-ExecutionPolicy Текущая политика выполнения
Области: {MachinePolicy|UserPolicy|Process|CurrentUser|LocalMachine}
Политики: {AllSigned|Bypass|Default|RemoteSigned|Restricted|Undefined|Unrestricted}
Get-ExecutionPolicy -List Политика выполнения каждой области
Get-ExecutionPolicy -Scope LocalMachine Политика выполнения в области LocalMachine

Git Cheat Sheet

Создать репозиторий

Создать новый локальный репозитарий

$ git init <project_name>

Загрузить существующий репозитарий

$ git clone [project_url]

Наблюдать за репозиторием

Список новых и модифицированных файлов не committed

$ git status

Вывести список изменений

$ git log

Вывести весь список изменений

$ git log --all

Вывести сокращенный список изменений

$ git log --oneline --decorate

Ветви

Список всех локальных ветвей

$ git branch

Список всех локальных и удаленных ветвей

$ git branch -av

Перейти на ветку

$ git checkout my_branch

Создать новую локальную ветку

$ git branch new_branch

Удалить ветку

$ git branch -d my_branch

Синхронизация

Список связанных внешних проектов

$ git remote

Связать с внешним проектом

$ git remote add <name> <URL>

Внесение изменений

Подготовка файла к commit

$ git add <file_name>

Подготовить все измененные файлы к commit

$ git add .

Commit всех проиндексированных файлов

$ git commit -m "Моё сообщение"

Изменить последний commit

$ git commit --amend

Отменить изменения, в сохранённых изменениях

$ git reset <file_name>

Посмотреть изменения

$ git show <name>

Только что внесенный изменения

$ git diff

OpenSSL Cheat Sheet

Общие команды

Получение справки о всех командах

openssl help

Получение справки по команде

openssl <command> -h

Получить версию программы

openssl version

Получить полное описание

openssl version -a

Генерация ключей

Генерация закрытого ключа

openssl genpkey -out fb.key -algorithm RSA

Показать структуру закрытого ключа

openssl pkey -in fb.key -text -noout

Показать открытую часть

openssl pkey -in fb.key -pubout

Запрос на создание сертификата

openssl req -new -key fb.key -out fb.csr

Просмотр запроса

openssl req -text -in fb.csr

Подписание сертификатов

openssl x509 -req -days 365 -in fb.csr -signkey fb.key -out fb.crt

Создать самозаверяющий сертификат

openssl req -new -x509 -days 365 -key fb.key -out fb.crt

Проверка сертификата

Проверка сертификата используя доверенный сертификат

openssl verify -CAfile cacert.pem www.example.org.pem

Проверка с выводом цепочки

openssl verify -show_chain -CAfile cacert.pem www.example.org.pem

Проверка цепочки сертификатов

openssl verify -CAfile cacert.pem -untrusted Intermediate.pem www.example.org.pem

Преобразование

PEM -> DER

openssl x509 -inform PEM -in fb.pem -outform DER -out fb.der

DER -> PEM

openssl x509 -inform DER -in fb.der -outform PEM -out fb.pem
