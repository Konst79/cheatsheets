Chocolatey cheat sheet

Просмотр

Список пакетов

choco list

Список локальных пакетов

choco list -lo

Информация о пакете

choco info <name_packet>

Поиск пакета

choco search <name_packet>

Установка

Установка пакета

choco install <name_packet>

Удаление

Удаление пакета

choco uninstall <name_packet>

Обновление

Обновление всех пакетов

choco upgrade all

Обновление пакета

choco upgrade <name_packet>
